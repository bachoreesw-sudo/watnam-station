1. สถาปัตยกรรมและการควบคุมระบบ
  • ระบบ Cloud Real-time: ข้อมูลถูกส่งจากบอร์ด ESP32 ผ่านโปรโตคอล HTTPS ไปยัง Firebase Database โดยมีความหน่วง (Latency) ต่ำกว่า 500ms ทำให้การแสดงผลบน Dashboard เกือบจะเป็นปัจจุบันที่สุด
  • การแสดงผลกราฟวิเคราะห์: กราฟเส้นสีฟ้า (Cyan Neon) ถูกออกแบบมาเพื่อวิเคราะห์แนวโน้ม (Trend Analysis) คุณสามารถซูมดูช่วงเวลาที่ค่า pH มีการแกว่งตัวผิดปกติเพื่อหาสาเหตุของมลพิษในน้ำได้
  • ระบบจำลองปลา 3D Twin: เป็นนวัตกรรมการแจ้งเตือนรูปแบบใหม่ โดย AI จะประมวลผลค่า pH และอุณหภูมิ แล้วสั่งการให้โมเดลปลาแสดงท่าทาง "ปกติ" หรือ "หงายท้อง" เพื่อให้ผู้ดูแลรับรู้ถึงวิกฤตได้ทันทีโดยไม่ต้องอ่านตัวเลข

3. การบำรุงรักษาฮาร์ดแวร์เซ็นเซอร์
  • pH Probe E-201-C

    หัววัด pH มีอายุการใช้งานและมีความไวสูง **ห้ามปล่อยให้หัวแก้วแห้งเด็ดขาด** ควรมีน้ำแช่อยู่ในฝาครอบเสมอ (น้ำยา KCL) หากค่าเพี้ยนให้ทำการ Calibrate ด้วยสารละลาย Buffer pH 4.0 และ 6.86
  • Ultrasonic JSN-SR04T

    เป็นเซ็นเซอร์วัดระยะทางแบบกันน้ำ (Waterproof) ควรติดตั้งให้ขนานกับผิวน้ำ และเช็ดคราบตะไคร่หรือหยดน้ำที่เกาะอยู่หน้าโพรบออกทุกเดือนเพื่อความแม่นยำ
  • DS18B20 Temperature

    เซ็นเซอร์ดิจิทัลที่ส่งข้อมูลแบบ One-Wire ควรตรวจสอบจุดต่อสายไฟ (Resistor 4.7k) ว่าไม่มีสนิมเขียวเกาะ เพื่อป้องกันสัญญาณรบกวน
ข้อควรระวังขั้นสูง

การวางสายสัญญาณของหัววัด pH ใกล้กับสายปั๊มน้ำแรงดันสูง (AC Pump) อาจทำให้เกิดสัญญาณรบกวน (Noise) อย่างรุนแรง ทำให้ค่า pH กระโดดไปมา แนะนำให้ใช้สายชีลด์ (Shielded Cable) หรือวางห่างจากแหล่งกำเนิดสัญญาณรบกวนอย่างน้อย 1 ฟุต

4. ระบบวัดความเค็ม (EC/TDS) ผ่านโมดูล RS485
  • การเชื่อมต่อฮาร์ดแวร์:
    เซ็นเซอร์ EC ต่อเข้ากับโมดูลขยายสัญญาณ RS485 จากนั้นต่อเข้ากับเครื่องวัดแรงดันไฟ RTU (12-24V) เพื่อแปลงสัญญาณ Analog ให้เป็นดิจิทัล และส่งข้อมูลผ่านขา A/B ของ RS485 เข้าสู่บอร์ด ESP32/ESP8266
  • การ Calibrate ค่าความเค็ม:
    ใช้น้ำทะเลมาตรฐาน (Salinity 30-35 ppt) หรือสารละลาย KCL มาตรฐาน วัดค่าและปรับแต่งค่า Offset ใน Code ของบอร์ดควบคุมให้ตรงกับค่ามาตรฐาน เพื่อความแม่นยำในการเลี้ยงปลากะพงในแม่น้ำบางนรา
การป้องกันการผุกร่อน

บริเวณที่ต่อสายไฟระหว่างเซ็นเซอร์ EC และโมดูล RS485 ต้องหุ้มด้วย Heat Shrink และเทรองต์น้ำยาง (Liquid Electrical Tape) ทุกชั้น เพื่อป้องกันน้ำกร่อยซึมเข้าไปกัดกร่อนวงจร
